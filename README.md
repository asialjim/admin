# 微应用管理系统

基于 Vue 3 + Nuxt.js + Element Plus 的后台管理系统框架

## 技术栈

- **前端框架**: Vue 3
- **构建工具**: Nuxt.js 3
- **UI组件库**: Element Plus
- **编程语言**: TypeScript

## 项目结构

```
admin/
├── pages/           # 页面文件
├── public/          # 静态资源
├── app.vue          # 应用入口
├── nuxt.config.ts   # Nuxt配置
└── package.json     # 项目依赖
```

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发环境运行

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 功能说明

这是一个基础的后台管理系统框架，已经集成了：
- Element Plus UI组件库
- 基本的页面结构
- 响应式布局
